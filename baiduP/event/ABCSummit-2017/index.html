这是变革的时代，技术的发展比任何时候都超乎想象！

无论传统产业，还是新兴行业，不管是否准备就绪，人工智能的时代已经来临！

无论是面向端的应用，还是企业级的平台，ABC（AI，Big Data，Cloud Computing）三位一体的融合已成为趋势！

技术进步如何推动企业变革，传统产业如何快速获得人工智能红利，新兴行业如何把握未来趋势。数据、模型、算法、平台、场景，如何有效应用！

为变革而来，百度2017ABC SUMMIT即将开启！

大会议程

◇主论坛

9:00-12:00

深入解读ABC（AI，Big Data，Cloud Computing）行业趋势，重磅发布最代表技术融合的产品，全面展示最代表技术能力的应用案例。

◇专题论坛 — ABC基础技术与开放平台

13:30-17:00

异构计算，边缘计算，智能存储，NewSQL，云原生架构等ABC （AI，Big Data，Cloud Computing）新兴技术在百度云全系列产品实践和解决方案；构建安全，高可用，可扩展的最佳实践案例。深度学习、计算机视觉、认知计算等AI基础技术及应用案例。

◇专题论坛 — 视频智能技术与平台

13:30-17:00

ABC（AI, Big Data, Cloud Computing）在视频理解方面的技术及案例，包括视频内容分析、视频内容审核和视频封面选图等AI产品和解决方案，如何提升视频内容生产、传播、变现的质量和效率。解读百度视频AI技术及内容生态如何赋能行业。

◇专题论坛 — 设备智能与物联网平台

13:30-17:00

从连接万物到唤醒万物，物联网正在经历着新的跨越。工业设备，车联网，智慧物流，智能建筑从设备到平台应用有了不一样的发展，聚焦ABC（AI, Big Data, Cloud Computing）与IOT的融合应用，从技术到平台，从场景构想到应用案例，全方位展示物联网2.0的发展。

◇专题论坛 — 大数据与智能营销

13:30-17:00

秉承开源开放，百度提供丰富的人工智能和大数据服务，分享在数字营销领域人工智能和大数据如何助力客户研究、产品推荐、业务分析、广告营销等方面的应用及最佳实践。

◇专题论坛 — 家居与汽车智能

13:30-17:00

人工智能，大数据，云计算在智能生活领域的最新技术与应用。涵盖智能家居、智能驾驶、智能设备等领域，语音、图像、NLP、深度学习等多项人工智能技术助力智能生活。

◇专题论坛 — 智慧金融与金融安全

13:30-17:00

ABC（AI, Big Data, Cloud Computing） 如何为金融行业客户带来安全、合规、且符合行业监管要求的云计算服务。智能金融解决方案如何为客户解决营销、客服、风控等问题，提升业务效率；第三方支付、P2P网贷、理财、众筹、传统金融典型应用场景分享。
